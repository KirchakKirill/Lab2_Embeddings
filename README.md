# Рекомендательная система на основе эмбеддингов (Content-Based)

**Вариант 4** | **Лабораторная работа по эмбеддингам**

## Цель лабораторной работы
Познакомиться с эмбеддингами.

## Ключевая идея
Эмбеддингами можно сравнивать похожесть текстов по смыслу.

## Формат работы
- Можно делать в группах до 5 человек при соответствующем качестве
- Творческий подход приветствуется
- Результат можно будет переиспользовать в следующих лабораторных

## Результат
Микросервис или CLI приложение для рекомендательной системы (content-based).

**Минимальное решение:** на вход описание нового фильма/игры, на выходе предлагаются похожие.

**Полное решение:** проведён анализ качества рекомендаций в зависимости от используемых функций расстояния и превращаемых в эмбеддинг данных (помимо описания можно включать другие данные).

## Технологический стек
**Эмбеддинг модели:** Ollama (локально через REST API)
**Векторная БД:** PostgreSQL + pgvector
**Стек разработки:** любой

- [x] Проведено сравнение разных функций расстояния

Результат предлагается оформить в виде гитхаб-репозитория.
